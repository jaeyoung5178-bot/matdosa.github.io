맛도사 홈페이지 (영문 파일명 버전) — 한글 파일명 404 문제 해결판
================================================================

[중요] 이번엔 파일 이름이 전부 영어예요. 한글 파일명이 깃허브에서
       404를 일으켜서, 영어로 바꿨습니다. 이 폴더 파일을 그대로 올리세요.

[이 폴더의 파일]
 ● index.html       ← 메인. (포트폴리오/허브 탭이 portfolio.html, hub.html 을 부름)
 ● portfolio.html   ← 포트폴리오 (필수)
 ● hub.html         ← 카페 탐방 허브 (필수)
 ● card.html        ← 명함(독립, 인쇄용)
 ● logo_full.png / logo_mark.png ← 로고 원본(보관용)

────────────────────────────────────────
[올리는 법 — 기존 파일 교체]
 1) 깃허브 저장소(matdosa.github.io) → Code 탭
 2) 기존에 잘못 올라간 한글 파일들은 그대로 둬도 되지만, 깔끔히 하려면
    삭제 권장: 맛도사_포트폴리오.html / 맛도사_카페탐방_허브.html 등
    (각 파일 클릭 → 오른쪽 ··· 또는 휴지통 아이콘 → Delete → Commit)
 3) Add file → Upload files → 이 폴더의 파일 6개를 끌어다 놓기
 4) Commit changes
 5) 1~2분 뒤, 시크릿창에서 사이트 열고 포트폴리오/허브 탭 확인
